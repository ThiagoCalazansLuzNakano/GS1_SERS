# GS1_SERS
# Sistema Inteligente de Monitoramento Energético - Missão Espacial Interativa

## Descrição
Este projeto consiste em uma solução computacional desenvolvida em Python voltada para o monitoramento e controle operacional de uma missão espacial experimental. Esta aplicação funciona como um **painel de controle interativo**, onde o operador insere os dados dos sensores (energia, temperatura e comunicação). 

O sistema aplica conceitos de sustentabilidade, eficiência energética e lógica de inteligência artificial introdutória (sistemas baseados em regras) para gerar alertas e tomar decisões automatizadas em cenários críticos.

## Funcionalidades
- **Interface Interativa via Terminal:** Coleta ativa de dados de telemetria (Temperatura, Nível de Bateria e Qualidade de Sinal).
- **Tratamento de Erros Robusto:** Validação de dados de entrada que impede o travamento do software caso o operador digite caracteres inválidos.
- **Geração Automática de Alertas:** Identificação instantânea e visual de anomalias térmicas, falhas de comunicação e escassez de energia.
- **Tomada de Decisão Automatizada (IA Introdutória):** Sistema lógico que altera o estado de operação da nave (como ativação de modo de eco-sustentabilidade para preservação de vida e desligamento de módulos secundários) com base nos dados inseridos.
- **Painel de Controle:** Interface de terminal organizada, de fácil leitura, separando dados, alertas e ações tomadas.

* **Eficiência e Sustentabilidade:** Algoritmo focado no gerenciamento inteligente e economia de energia renovável (solar) em momentos de crise energética espacial.
